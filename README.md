## Comandos Git
- ls &lt;archivo&gt;
- git status
- git status --short
    {>  ?? - Untracked files
    {>  A - Files added to stage
    {>  M - Modified files
    {>  D - Deleted files

- git add &lt;archivo&gt;
- git add --all / git add -A
- git commit -m "Mensaje"
- git commit -a -m "Mensaje"
